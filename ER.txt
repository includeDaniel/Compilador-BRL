Expressoes Regulares - Linguagem BRL:

ID       => [a-zA-Z_][a-zA-Z0-9_]*   (max 512 caracteres, case-sensitive)
INTEIRO  => [0-9]+
REAL     => [0-9]+\.[0-9]+
CONST_STR=> "[^"\n]*"
LOGICO   => verdadeiro | falso

Palavras Reservadas:
  Tipos   : inteiro  real  logico  caractere
  Comandos: se  senao  enquanto  faca  inicio  fim  leitura  escrita
  Outros  : ou  div  mod  verdadeiro  falso

Operadores:
  Relacionais : ==  <>  <  >  <=  >=
  Aritmeticos : +  -  *  /
  Logicos     : &&  ou
  Atribuicao  : :=

Delimitadores: ;  :  ,  (  )  [  ]  {  }

Comentarios: /* ... */  (podem abranger multiplas linhas)

Erros lexicos:
  - Caractere invalido (nao pertence ao conjunto permitido)
  - String nao fechada antes de quebra de linha ou EOF
  - Real com ponto sem digitos apos (ex: 3.)
  - Comentario nao fechado (/* sem */)
  - Identificador com mais de 512 caracteres
